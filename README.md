# Save The Princess

A 2D platformer game made in Unity.

## 🎮 Play Online (Demo)
[Play in Browser](https://filipzvkvc.itch.io/save-the-princess)

## 💾 Download
[Download Installer](https://github.com/filipzvkvc/SaveThePrincess/releases/tag/v0.1)

## 🕹️ Controls
- **Left/Right Arrow Keys** - Move
- **Up Arrow** - Jump
- **Down Arrow** - Crouch
- **Escape** - Pause
